# 建筑材料不燃性试验仿真系统 — 启动教程

---

## 一、首次使用（环境准备）

### 1. 安装 .NET 8.0 运行时

下载并安装 [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)（选择 Windows x64 版本）。

> 如果已安装过 Visual Studio 2022 或 .NET 8 SDK，可跳过此步骤。

### 2. 验证安装

打开 **命令提示符**（Win+R → 输入 `cmd` → 回车），输入：

```
dotnet --version
```

如果显示版本号（如 `8.0.x`），说明安装成功。

---

## 二、启动程序

### 方法一：双击启动（推荐，最简单）

1. 打开文件夹：`技能实训\源代码\NctSim\bin\Debug\net8.0-windows\`
2. 双击 **`NctSim.exe`** 即可启动

### 方法二：命令行启动

打开 **命令提示符**，依次输入：

```
cd /d "D:\java_workspace\技能实训\源代码\NctSim"
dotnet run
```

> 如果提示编译错误，先执行 `dotnet build`，再执行 `dotnet run`

## 三、操作流程

### 1. 登录

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | `admin` | `123456` |
| 试验员 | `experimenter` | `123456` |

选择角色 → 输入密码 → 点击 **「登录」**

### 2. 新建试验

点击 **「新建试验」** → 填写样品信息：

- 样品编号（如 `PROD-001`）
- 样品名称
- 规格、直径、高度
- 试验前质量、环境温度、环境湿度

→ 点击 **「确定」**

### 3. 开始升温

点击 **「开始升温」** → 炉温从 25°C 开始上升到 750°C

- 升温过程约 **2.5 分钟**
- 右侧曲线图可看到温度实时上升
- 状态显示 **「升温中」**（橙色）

### 4. 等待就绪

温度达到 750°C 并稳定后，自动变为 **「就绪」** 状态（绿色）

此时 **「开始记录」** 按钮会启用

### 5. 开始记录

点击 **「开始记录」** → 状态变为 **「记录中」**（蓝色）

- 系统开始计时并持续记录温度数据
- 曲线图显示样品表面温度和中心温度的变化

### 6. 停止记录

点击 **「停止记录」** → 状态变为 **「完成」**（绿色）

### 7. 保存记录

点击 **「保存记录」** → 填写试验后质量等数据 → 点击确定

- 系统自动保存数据到数据库
- 自动生成 **Excel 报告** 和 **PDF 报告**

---

## 四、查看历史记录

切换到 **「记录查询」** 页签：

1. 设置日期范围
2. 点击 **「查询」**
3. 双击某条记录可查看详情
4. 选中记录后可点击 **「导出 Excel」** 或 **「导出 PDF」** 重新生成报告

## 六、常见问题

**Q: 双击 exe 没反应？**
A: 检查是否安装了 .NET 8.0 Desktop Runtime。

**Q: 温度没变化？**
A: 先点击「新建试验」，再点击「开始升温」。

**Q: 「开始记录」按钮灰色点不了？**
A: 需要等待温度升到 750°C 并稳定（约 2.5 分钟），状态变为「就绪」后才能开始记录。

**Q: 保存时报错？**
A: 请确认已停止记录（状态为「完成」）后再保存。
